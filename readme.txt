Wordpress shortcode for generate jdownloader Click'N'Load


== Description ==

This plugin implement the click n load v2 from jdownloader

more info here: http://jdownloader.org/knowledge/wiki/glossary/cnl2

For usage only put the shortcode:

```
[jdownloader]
https://mega.nz/#URL
https://mega.nz/#URL
[/jdownloader]

```

And all your urls should encrypted with a form.


The button will deactivate because you need jdownloader running!


== Installation ==

Install like other plugin 
e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
